## Contributing

How can I contribute?

1. Clone the repo.
   ```sh
   git clone https://github.com/hack-as-a-service/frontend
    ```
2. Change directory.
   ``` cd frontend
   ```
3. Install dependencies. 
   ```
   yarn install
   ```
   
4. Start the server.
   ```
   yarn dev
   ```
